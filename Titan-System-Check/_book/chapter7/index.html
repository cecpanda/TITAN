第 7 章 主要进程

主要进程的确认

将TITAN下运行的主要进程的执行状况以界面显示。

从该输出结果，可以调查出经常执行的进程是否出现出现停止的现象。

  • 正在运行的进程显示为：Normal.。

  • 进程停止时，出现以下信息:

    [PID File (<进程名>.pid) is invalid. Process is not running.]
